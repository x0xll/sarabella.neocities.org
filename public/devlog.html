Last modified: February 11th, 2025

Here is the current state of the archiving of the Bella Sara website games:

We have successfully managed to locate and restore several of the original Flash games from the Bella Sara website. These include Cloud Jumper, Treasure's Hunt, Jigsaw Puzzle, Coloring Game (Fiona only), Memory Game, Santo's Quiz, the Baby Bella ad, and the old version of the World Map.

Additionally, we have partially recovered or restored the files for several other games, such as the Art Studio, Lance Riding, Storybook, My Horse (sprite only), Firelight Festival, and the newer version of the World Map. These games are still undergoing work to fully restore their original functionalities, and many games are missing files (e.g., the Lance Riding game is missing the files for the later levels).

Unfortunately, some games remain missing or seem to be permanently broken (e.g., Bellaâ€™s Beauty Box). To address this, we have started recreating a few of these games using the Phaser framework. As of this update, the Stables, the Spectacular Jumping Game and Belo's Quizare all available to play, although still as a work in progress. We are also currently working on recreating Citrustacked, Bellapedia, and Bellisimo's Jumping Contest, which should all hopefully be available soon.

Beyond restoring the original games, we have also started creating custom games inspired by the ones from the website. Currently, two are available: a Random Art Idea Generator, a Dress-Up Game based on the Stables and an Adventures Quiz and Autumn Journey Quiz based on Belo's Quiz.

We do hope to eventually restore or recreate all the games from the site if we can, but this will obviously take time. If you would like to help us out or would like a more detailed overview of this project, we would recommend starting here: Bella Sara Archival Notes. This project is a community effort and every little bit helps!
